# AnonymousMessage Android Client

## Overview
Android client for AnonymousMessage built with Java, featuring secure messaging through TOR network.

## Features
- Secure messaging via TOR
- Voice and video calls
- Screen sharing capabilities
- Location sharing
- Group messaging
- User authentication with email verification
- Avatar management

## Security
- End-to-end encryption
- Anonymous routing through TOR
- Local data encryption
- Secure authentication mechanisms
