# AnonymousMessage PC Client

## Overview
Desktop client for AnonymousMessage built with C++, featuring secure messaging through TOR network.

## Features
- Secure messaging via TOR
- Voice and video calls with camera support
- Advanced screen sharing (full screen or specific applications)
- Location sharing
- Group messaging
- User authentication with email verification
- Avatar management
- macOS-style glass design interface

## Security
- End-to-end encryption
- Anonymous routing through TOR
- Local data encryption
- Secure authentication mechanisms
